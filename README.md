Parkinson's Disease Prediction Model

## Overview
This project predicts Parkinson's Disease using machine learning and deep learning techniques based on EEG and clinical data.

## Features
- Data preprocessing and feature extraction
- Machine Learning model (.sav)
- Deep Learning CNN model (.h5)
- Performance evaluation using accuracy and loss metrics
- Parkinson's Disease prediction interface

## Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn
- TensorFlow/Keras
- Matplotlib
- Streamlit

## Files
- app.py : Main application
- Notebook.ipynb : Model training notebook
- final_dataset_parkinson.csv : Dataset
- model.sav : Trained ML model
- tff_model.h5 : Deep Learning model

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
